To use this application, you can log in as a student or an administrator. Each of these will take you to a different landing page. Lecturers will be included in the future.

To log in as an administrator, apply user-name "adminUser" with password "admin".

To log in as a student, check the console on startup to see a number of student user IDs. To construct each student's password, simply prepend the user ID with "pw".

For example:

  • Julie Banks: user-name = "jebs", password = "pwjebs".
  • Ashara Singh: user=name = "aash", password = "pwaash".
  • Richard Hope: user-name = "rdhe", password = "pwrdhe".
